Installing MetaPrint2D

MetaPrint2D is a set of plugins for Bioclipse. There are two ways to obtain a Bioclipse-MetaPrint2D package:

Download a pre-built Bioclipse-MetaPrint2D package

Please see links available from the MetaPrint2D wiki page.

Download Bioclipse and use the software update option

This guarantees the latest version of Bioclipse and MetaPrint2D.

Bioclipse is available from www.bioclipse.net, see the installation page for more information.

After installing Bioclipse, start it and go to the menu Help > Software Updates. Check MetaPrint2D and press finish, and after restart you have MetaPrint2D installed.
