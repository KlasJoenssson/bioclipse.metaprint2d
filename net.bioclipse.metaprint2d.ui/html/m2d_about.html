About MetaPrint2D

Overview

MetaPrint2D is a tool to predict metabolic sites in chemical structures based on historic metabolic data, described by circular fingerprints.

If you use MetaPrint2D in Bioclipse, please cite:

Lars Carlsson, Ola Spjuth, Samuel Adams, Robert Glenn, Scott Boyer
Use of Historic Metabolic Biotransformation Data as a Means of Anticipating Metabolic Sites Using MetaPrint2D and Bioclipse
BMC Bioinformatics 2010, 11:362 doi:10.1186/1471-2105-11-362
